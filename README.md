> Um mini-programa introdutório feito em Java puro para ler entradas do usuário via teclado e demonstrar a aplicação prática de operadores matemáticos, relacionais e lógicos. 💻

## ✨ Funcionalidades Demonstradas

O projeto recebe dois números informados pelo usuário no terminal e aplica testes demonstrando:

- **➗ Operadores Aritméticos:** Soma (+), Subtração (-), Multiplicação (*), Divisão (/) e Resto da divisão (%).
- **⚖️ Operadores Relacionais:** Maior (>), Menor (<), Igual (==) e Diferente (!=) para comparação entre os valores.
- **🧠 Operadores Lógicos:** Testes com `AND` (&&), `OR` (||) e Negação/Inverso (!).
